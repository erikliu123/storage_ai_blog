## 简介

cs336是一个专注于存储系统、计算机科学和人工智能领域的数据模型项目。该项目通过静态数据文件的形式，组织和管理各类技术内容，包括学术论文、研究团队信息、会议数据、存储故障分析、开源项目等。数据模型设计遵循类型安全、可扩展性和易维护性的原则，为前端界面提供结构化的数据支持。

**更新** 新增OSDI 2025和ATC 2024会议论文数据结构增强，提供完整的论文解读、架构分析和性能数据展示功能，包括改进的描述格式和增强的架构图支持。FAST会议档案系统经过重大改进，添加了paperUrl字段，替换虚构论文为GogetaFS、PolyStore、Integrity Checking、MedFS等真实研究工作的详细数据结构。**新增** ArchivePaperData接口扩展，新增paperUrl字段支持，增强论文真实性验证和溯源能力。FAST档案系统现在包含五个详细的学术论文部分，每个论文都包含完整的问题背景、技术设计、对比分析、性能评估和局限性说明章节，显著增强了学术论文的完整性和实用性。

## 详细组件分析

### 论文数据模型

论文数据模型是项目的核心，包含了丰富的元数据和内容结构：

#### 数据结构设计原则

1. **国际化支持**：同时支持中英文标题和描述
2. **多来源兼容**：统一处理不同数据源（DBLP、arXiv、微信公众号）
3. **内容丰富性**：支持图片、章节、架构图等多种内容形式
4. **分类体系**：通过枚举类型确保分类的一致性
5. **真实性验证**：通过paperUrl字段链接到原始论文，确保数据真实性

#### 字段定义和验证规则

| 字段名 | 类型 | 必填 | 验证规则 | 描述 |
|--------|------|------|----------|------|
| id | string | 是 | 唯一性、格式约束 | 论文唯一标识符 |
| title | string | 是 | 长度限制 | 论文标题 |
| titleZh | string | 否 | 可选中文标题 | 中文标题 |
| authors | string[] | 是 | 非空、数组验证 | 作者列表 |
| venue | string | 是 | 枚举验证 | 发表会议或期刊 |
| year | number | 是 | 数值范围验证 | 发表年份 |
| date | string | 是 | 日期格式验证 | 发表日期 |
| category | Category[] | 是 | 枚举验证 | 分类标签数组 |
| abstract | string | 是 | 长度限制 | 摘要内容 |
| coreContributions | string[] | 是 | 非空验证 | 核心贡献列表 |
| tags | string[] | 是 | 非空验证 | 标签数组 |
| url | string | 是 | URL格式验证 | 原始链接 |
| source | Source | 是 | 枚举验证 | 数据来源 |
| sourceLabel | string | 是 | 格式验证 | 来源显示标签 |
| readTime | number | 是 | 数值范围验证 | 阅读时间（分钟） |
| isNew | boolean | 否 | 布尔验证 | 是否为新论文 |
